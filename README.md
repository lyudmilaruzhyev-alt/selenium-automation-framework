# Selenium Automation Framework

Automation testing framework built using Java, Selenium WebDriver, and TestNG.

## Features
- Page Object Model (POM) design
- Data-driven testing using TestNG DataProvider
- Positive and negative test scenarios
- TestNG XML suites (Smoke, Regression)
- Explicit waits (WebDriverWait)

## Test Coverage
- Login functionality (positive & negative)
- Validation of error messages
- End-to-end test scenarios

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## Project Structure
- BasePage / BaseTest
- Page classes (LoginPage, SuccessfulLoginPage)
- Test classes
- TestNG XML suites

## How to Run Tests
mvn clean test
